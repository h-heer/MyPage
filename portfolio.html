AWS Security Notification System

  • In this project, our team created a security system leveraging AWS to enhance organizational security. The system was designed to notify the organization administrator immediately upon triggering any security alert. We developed a website using the Django framework, featuring a login page for secure access.

  • The system flagged security events by scanning user activity logs stored in Amazon S3 buckets. An EC2 instance was used to monitor these logs for suspicious activity or predefined security events. When such an event was detected, the system automatically sent notifications to the administrator via email or SMS, enabling rapid response and mitigation.
