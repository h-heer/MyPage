Uber NYC Visulaization

  • Role: Data Analyst
  • Technologies & Frameworks: Python, Spark, Docker, Tableau, AWS S3

  • In this project, I constructed an end-to-end data pipeline to analyze Uber rides in New York City. I began by ingesting raw trip data from public sources and storing these datasets in Amazon S3 as a centralized data lake. Using Python scripts, I merged and standardized multiple files, carefully validating key fields like timestamps, fare amounts, and geolocation data. For large-scale processing and data transformations, I leveraged Apache Spark to filter out anomalies, aggregate critical metrics, and partition outputs into efficient formats such as Parquet. To ensure a uniform environment across development and production, I incorporated Docker for containerizing Python and Spark jobs.

  • After generating a reliable, cleaned dataset, I connected the results to Tableau to build interactive dashboards—highlighting time-based demand trends, geographic pickup hotspots, and seasonal usage patterns. This setup enabled timely insights into ride frequency and revenue opportunities, ultimately guiding strategic decisions for Uber’s operations in NYC. By combining modern data engineering practices with robust analytics and visualization, I provided a scalable, maintainable solution that transforms raw data into actionable intelligence for stakeholders.

AWS Security Notification System

  • Role: Team Lead
  • Technologies & Frameworks: AWS EC2, S3, SNS, IAM, Django, Email/SMS APIs
  • In this project, I played a role in developing an AWS-based security notification system designed to immediately alert administrators about critical security events. My team built a Django-powered web application, implementing secure authentication and role-based access for administrators. The solution scanned user activity logs stored in Amazon S3 and employed an EC2 instance that continuously monitored these logs for suspicious activity. Whenever anomalies or predefined security triggers were detected, the system automatically dispatched alerts through AWS SNS (email & SMS) to ensure rapid response and threat mitigation. This project bolstered organizational security by reducing response times and safeguarding sensitive assets.
